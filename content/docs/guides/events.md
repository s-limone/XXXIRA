## Daily Checklist

Do these every day for steady, compounding progress:

- [ ] Collect from all city production buildings
- [ ] Complete Daily Jobs / Daily Quests (free resources and rewards)
- [ ] Attack monsters on the World Map (Valor, equipment drops)
- [ ] Run at least one Crypt exploration (Tar permitting)
- [ ] Check for active clan tasks or donation requests
- [ ] Check for any active limited-time events and progress their milestones

## Daily Jobs & Achievements

Daily Jobs are short tasks that reset every 24 hours. They reward Silver, Gold, resources, and event points. Even completing a portion of the daily list provides meaningful progression — make it a habit.

Achievements are long-term milestones that reward large bonuses once completed. Focus on achievements that align with your current playstyle (monster kills, crypt completions, etc.).

## Epic Monster Events — March Format

Different events require different march compositions. Check which captains each event accepts:

| Event | Who leads the march |
|-------|-------------------|
| **Doomsday** | One Captain |
| **Arachne's Swarm** | All three Captains |
| **Armageddon** | Hero only |
| **Hellforge** | One Captain |
| **Shadow Invasion** | Hero + all Captains |

> Always use proper **troop stacking** for any Epic Monster event — see [Combat & Battles](/docs/guides/combat). The 2:1 pyramid dramatically reduces revival costs.

## Trials of Olympus

Five sequential daily trials that run as a clan tournament:

| Day | Trial | Format | Troop Notes |
|-----|-------|--------|-------------|
| 1 | **Chimera** | Clan Raid — one or more captains | Mounted/flying Guardsmen, mercenaries, monsters |
| 2 | **Basilisk** | Hero-led solo march | Same troop composition |
| 3 | **Lava Fountains** | PvP zone — 90% resurrection discount | Hold Oil; you keep 100% even when attacked |
| 4 | **Tartaros Crypts** | Crypt runs using Lava Oil | Gear collection phase |
| 5 | **Briareus** | Up to 3 captains | Stacked troops + mercenaries + monsters |

**Recommended captains for Chimera / Briareus:** Beowulf, Hercules, Cleopatra, Amanitore.

## Event-Specific Tips

- **Arachne's Swarm** — best event for levelling captains cheaply; not recommended if Valor is the priority
- **Armageddon** — best for hero XP; caution if you're running a focused Silver Farm account (over-levelling hero may hurt)
- **Hellforge coins** persist between events — don't abandon the event midway or you waste accumulated coins
- **Shadow Invasion** — highest Conquest Point source; prioritise during Clash for the Throne preparation
- More kills = more tournament points, Valor, and XP. Keep attacking even if a top finish is out of reach.

## Tips for Maximising Events

- Read the event rules before spending resources — some events reward specific troop types
- Save Gold and speedups for major events with the best milestone rewards
- Coordinate with the clan during clan-vs-clan events
- Stack troops correctly for any event where troops die — see [Combat & Battles](/docs/guides/combat)
